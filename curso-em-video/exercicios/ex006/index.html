Tags Obsoletas

Há algumas tags em HTML5 que, mesmo que fossem utilizadas no passado, encontram-se, atualmente, em desuso, em não conformidade.
Esse fenômeno ocorre pela busca de tornar o HTML uma linguagem voltada para a semântica do site, enquanto que os demais componentes ficam a cargo do CSS, do JavaScript, etc.

As tags obsoletas podem ser encontradas no link a seguir: https://dev.w3.org/html5/pf-summary/obsolete.html
