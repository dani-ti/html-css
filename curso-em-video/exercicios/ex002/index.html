Exercício 2

Pode-se escrever um parágrafo de qualquer maneira, basta colocar o conteúdo entre as tags "<p>" e "</p>".


Caso seja necessário quebrar o texto em algum lugar, utilize a tag "<br>".

Em códigos profissionais, usar várias tags "<br>" consecutivamente, para fazer maiores espaçamentos, é errado. Para isto, deve-se recorrer ao CSS.

Tags "<br>" podem assumir um valor menor (1) ou um valor maior (2) de espaçamento.
(1): Consegue ser obtido ao se utilizar "<br>" dentro das tags "<p>" e "</p>".
(2): Consegue ser obtido ao se utilizar "<br>" fora das tags "<p>" e "</p>".

Entre dois parágrafos diferentes, já existe um espaçamento adotado.
