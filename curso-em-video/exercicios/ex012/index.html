Tipos de CSS

CSS Inline

A estilização é efetuada na mesma linha em que está a tag. É utilizada, por tanto, para estilizações pontuais.


CSS Interno/Local

A estilização é efetuada no mesmo arquivo em que está o código do site estlizado.


CSS Externo

A estilização é efetuada num arquivo diferente, em que está somente a estilização do CSS para um ou mais sites.


Hierarquia de CSS

Ao recorrer a mais de um tipo de estilização, esta ocorrera conforme a seguinte ordem de prevalência: CSS Inline > CSS Interno > CSS Externo.
