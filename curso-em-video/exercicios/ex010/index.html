Links

Links Externos

Apontam para uma página dentro de outro servidor.

Exemplos:
YouTube
YouTube
Fuvest

Links Internos

Apontam para uma página dentro do mesmo servidor.

Exemplos:
Segunda Página
Terceira Página

Link para Download

O "type" do link, que é determinado pela extensão do arquivo que será baixado, pode ser encontrado em Media Types

Exemplo:
Que é Esclarecimento? de Immanuel Kant, em PDF.
Que é Esclarecimento? de Immanuel Kant, em WinRAR Archiver.
